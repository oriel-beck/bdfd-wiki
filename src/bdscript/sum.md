# $sum
Returns the addition of the provided numbers.

## Syntax
```
$sum[Number;...]
```

### Parameters
- `Number` `(Type: Integer, Float || Flag: Required)`: The numbers to add. Separate multiple values using `;`.

## Example
```
$nomention
Result: $sum[5;6;1]
```
