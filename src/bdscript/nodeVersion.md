# $nodeVersion
Returns the version of the current node.

## Syntax
```
$nodeVersion
```

## Example
```
$nomention
Node version: $nodeVersion
```

``` discord yaml
- user_id: 803569638084313098

```

> You can use [`$botNode`](./botNode.md) to find out which node your bot is on.
