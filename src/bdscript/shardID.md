# $shardID
Returns the shard ID of the current guild.

## Syntax
```
$shardID
```

## Example
```
$nomention
Shard: $shardID
```

``` discord yaml
- user_id: 803569638084313098
  username: RainbowKey
  color: "#E67E22"
  content: |
    !example
- username: BDFD Support
  user_id: 1009018156494368798
  color: "#378afa"
  bot: true
  verified: true
  content: |
    Shard: 1
```
