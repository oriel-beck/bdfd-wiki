# $toUppercase
Converts the provided text to uppercase format.

## Syntax
```
$toUppercase[Text]
```

### Parameters
- `Text` `(Type: String || Flag: Emptiable)`: The text to convert.

## Example
```
$nomention
$toUppercase[$message]
```
